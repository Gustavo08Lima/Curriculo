Projeto Cisco



2021 - 2021


Projeto: Implementação de Infraestrutura de Rede Segura para Multi-filiais Este projeto consistiu na criação de uma infraestrutura de rede robusta e segura para interligar as filiais de uma empresa nas cidades de São Paulo, Rio de Janeiro, Belo Horizonte e Salvador. Utilizando o software de simulação Cisco Packet Tracer, foram configuradas as redes locais (LANs) de cada filial e estabelecidas as conexões entre elas. Um dos pilares do projeto foi a implementação de uma DMZ (Zona Desmilitarizada) para hospedar serviços críticos, como um servidor web, isolando-os da rede interna e garantindo maior segurança. Os principais desafios superados foram:

  • A correta configuração do DHCP para que todos os dispositivos nas filiais recebessem endereços IP automaticamente a partir de um servidor centralizado na DMZ.

  • A correta configuração do DHCP para que todos os dispositivos nas filiais recebessem endereços IP automaticamente a partir de um servidor centralizado na DMZ.

O resultado foi uma rede simulada, funcional e segura, demonstrando a aplicação prática de conceitos avançados de roteamento, switching e segurança de rede.

Site de apoio ao Rio Grande do Sul



2021 - 2021


Este projeto consiste em um portal informativo e de arrecadação desenvolvido para auxiliar as vítimas das enchentes no Rio Grande do Sul.
Desenvolvido com HTML, CSS, JavaScript e Figma, a plataforma tem como objetivo centralizar informações sobre as causas, impactos e formas de contribuir para a recuperação da região afetada.

Principais Características:

Tecnologias Utilizadas: O front-end foi construído com HTML e CSS para estruturação e estilização, e JavaScript para interatividade. O design da interface foi prototipado previamente no Figma, garantindo uma experiência intuitiva e responsiva.

Conteúdo: Aborda as causas da tragédia (condições climáticas extremas, fatores humanos e mudanças climáticas), os impactos (comunidades desabrigadas, prejuízos econômicos e danos ambientais) e orienta sobre como ajudar através de doações, voluntariado e divulgação.

Funcionalidades: Inclui um sistema de cadastro de usuários (com validação de e-mail e senha) para potencializar o engajamento e um espaço dedicado a depoimentos que reforçam a solidariedade.

Objetivo Final: Servir como uma ferramenta digital acessível e eficaz para mobilizar apoio imediato e conscientizar a sociedade sobre a importância da reconstrução da região

Banco de dados



2021 - 2021


Plataforma de Gestão de Competições de eSports

Desenvolvimento de um sistema completo para gerenciar competições de eSports, permitindo:

  • Cadastro diferenciado de jogadores (com gêneros de jogo) e mentores (com formação e experiência)
  • Formação de equipes com jogadores e mentores
  • Criação de competições com premiação, datas e formato (online/presencial)
  • Registro detalhado de partidas com estatísticas, MVP e funções táticas
  • Banco de dados relacional com tabelas normalizadas para usuários, equipes, competições e partidas
  • Interface web desenvolvida com HTML, CSS, JavaScript e prototipada no Figma


Objetivo: Automatizar e organizar o ecossistema de competições de eSports, conectando jogadores, equipes e organizadores em uma plataforma centralizada
